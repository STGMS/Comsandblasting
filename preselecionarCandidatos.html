← Volver a Selección y Vinculación

Requisición De Personal

Formato de solicitud oficial para la necesidad de personal en la organización. Documenta el departamento solicitante, justificación de la necesidad, características del cargo, cantidad de vacantes y plazo de cobertura requerido para iniciar el proceso de selección. Este documento es fundamental para autorizar el inicio del reclutamiento.

Recepción De Hojas De Vida

Procedimiento y formatos para la recopilación de documentación de candidatos. Establece los requisitos de documentación, criterios de aceptación, fechas de cierre de convocatoria y mecanismos de organización de candidatos inscritos en el proceso de selección. Garantiza orden y trazabilidad en la gestión de aplicantes.

Lista Verificación Perfil De Cargo

Checklist para verificar que los candidatos cumplan con el perfil requerido. Facilita la evaluación objetiva de requisitos mínimos como educación, experiencia, competencias técnicas y especializaciones necesarias para avanzar en el proceso de selección.

Entrevista Preliminar

Guía y formato para la primera entrevista de filtrado de candidatos. Proporciona preguntas estructuradas para evaluación inicial, valoración de motivación, disponibilidad, expectativas salariales y compatibilidad con la cultura organizacional. Permite descartar candidatos no alineados con requisitos básicos.

Reclutamiento Interno

Protocolo para identificar y promover talento disponible dentro de la organización. Establece los procesos, criterios y beneficios para que empleados actuales puedan optar por nuevas posiciones, fomentando el desarrollo profesional interno y la retención del talento existente.

Reclutamiento Externo

Estrategia para atraer candidatos calificados del mercado laboral externo. Define canales de difusión, plataformas de publicación, criterios de publicidad de vacantes y metodologías para alcanzar el mayor número de candidatos potencialmente idóneos disponibles en el mercado.
